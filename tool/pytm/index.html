pytm

Kurzbeschreibung

pytm ist ein auf Python basierendes Threat Modeling Tool, welches 2020 entwickelt wurde und Threat Modeling automatisierter und von Entwicklern nutzbar zu machen. Hierbei wird von Entwicklern eine Beschreibung des zu analysierenden Systems bereitgestellt. Auf dieser Basis erstellt pytm, je nach Bedarf, Datenflussdiagramme, Sequenz-Diagramme und gibt Informationen zu potenziellen Threats aus.

Webseite:www.github.com/izar/pytm

Vor- & Nachteile

Vorteile

kostenlos
automatische Generierung von Datenflussdiagrammen, Sequenz-Diagrammen und Threat-Analysen
Einbindung in automatisierte build-/release-Pipelines möglich
Text-basiertes Modellieren von Systemarchitektur, d.h. Einbindung in Source Code Management Systeme und somit Revisionierbarkeit
Beschreibung individueller Threats möglich

Nachteile

nicht für das Betriebssystem Windows erhältlich
Beschreibung von Threats erfolgt in der Programmiersprache Python (kann Hürde darstellen, wenn Python nicht in Technologie-Stack enthalten oder gewünscht)

Abgleich Kriterien

Abbildung agiler Arbeitsmethoden möglich (User Stories, Sprints etc.)
Toolchain-Integration
Unterstützung von Arbeitsteiligkeit auf Personen-/Teamebene
Nicht-proprietäre interoperable Datenhaltung (z. B. Text-Formate)
Kostengünstig/ kostenlos erhältlich
Konfigurierbarkeit
Geringe Einstiegshürde
Mögliche BetriebssystemeLinux, MacOS
Bereitstellung von Templates für Threat-Modeling-Methoden
